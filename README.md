# 🏮 关帝灵签

> 心诚则灵 · 有求必应

一个基于关帝灵签100签的在线求签网页应用，纯前端实现，无需服务器，开箱即用。

## ✨ 功能特点

- 🎯 **随机求签** - 点击按钮即可获得一支灵签
- 📖 **指定查签** - 输入签号查询具体签文
- 📜 **求签记录** - 自动保存历史记录（本地存储）
- 🎨 **精美界面** - 古风设计，沉浸式体验
- 📱 **响应式布局** - 手机/电脑均可流畅使用
